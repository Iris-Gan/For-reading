# 笔 谈 · BITAN

> 上传书籍，邀请一位学者，在字里行间展开笔谈。

「笔谈」是一个 AI 共读应用：上传 PDF 或 TXT 电子书后，选择一位你欣赏的学者（教授、哲学家、作家等）作为共读伙伴，在阅读过程中对任意句子写下批注，AI 会以该学者的视角、思想和语气回应你的思考——像真正与一位学者并肩读书一样。


---

## ✨ 主要功能

- **电子书阅读**：支持上传 PDF / TXT 文件，自动按章节解析并分段呈现
- **学者模拟**：输入学者姓名即可召唤 AI 共读伙伴；也可粘贴 / 上传其论文、维基百科等资料，让 AI 更贴近该学者的思想风格
- **句子级批注**：用鼠标拖选任意文字片段进行高亮，在右侧气泡中写下你的批注
- **AI 回应**：学者以对话气泡形式回应你，引用相关理论拓展讨论
- **笔谈沉淀**：所有批注永久保留在原文旁，切换章节后可随时回看；目录中显示每章的批注数量
- **纸质质感**：米白色纸张纤维背景，衬线中文字体，模拟真实阅读体验

## 🎨 交互设计

### 双栏布局（6:4）

```
┌──────────────────────────┬──────────────────┐
│   原文（60%）            │   笔谈区（40%）   │
│                          │                  │
│   点击并拖选文字          │   ← 我的批注     │
│   ═══════════            │                  │
│   高亮片段 ▸ ▸ ▸         │   学者回复 →     │
│                          │                  │
└──────────────────────────┴──────────────────┘
```

### 气泡方向

- **我的批注**：左上角尖角气泡，从左侧滑入，米黄色底
- **学者回复**：右上角尖角气泡，从右侧滑入，浅绿色底

### 高亮视觉

- 选中文字时呈现明亮黄色高亮 `#fce68a`，带柔和光晕
- 已提交批注的句子保留浅黄色高亮作为标记
- 气泡内自动引用你所选的原文片段

## 🚀 使用方法

### 方式一：本地浏览器打开（推荐快速体验）

1. 下载 `bitan.html`
2. 双击在浏览器中打开即可使用
3. 点击"使用演示"按钮可直接进入本雅明《单行道》+ 阿多诺教授的共读场景

> ⚠️ 本地打开时 UI 所有功能正常，但 AI 教授回复需要 Anthropic API 环境支持，会显示"网络繁忙"。如需完整体验 AI 对话，请使用方式二或方式三。

### 方式二：部署到 Claude Artifacts

将 `bitan.jsx` 代码粘贴到 claude.ai 的 Artifacts 环境中运行，API 调用会自动生效。

### 方式三：集成到 React 项目

```bash
# 在你的 React 项目中
npm install react react-dom
```

将 `bitan.jsx` 作为组件导入即可：

```jsx
import BiTanApp from './bitan.jsx';

function App() {
  return <BiTanApp />;
}
```

## 💡 使用建议

### 选择什么样的学者

最有趣的共读体验来自**学者与书的"错位"或"对话关系"**：

- 让阿多诺读本雅明《单行道》——法兰克福学派内部的真实对话
- 让福柯读《规训与惩罚》之外的小说——观察权力分析如何延伸
- 让鲁迅读当代网络文学——跨时代的冷峻目光
- 让自己喜欢的老师读你正在研究的领域经典

### 如何写出好的批注

- 别只是复述原文，提出你的**困惑、联想、反驳或共鸣**
- 引用段落中的具体意象或概念，学者会针对性回应
- 写下"这让我想到……"、"但如果反过来看……"这样的开头，更容易激发深度对话

### 提升 AI 拟真度

在创建学者时，粘贴以下内容会显著提升回复质量：
- 学者的代表作目录
- 核心概念 / 术语清单
- 一段他们的原文片段作为风格参考
- 维基百科的学术简介
